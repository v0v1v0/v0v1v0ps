ref_taxa-class R Documentation

Class 'ref_taxa'

Description

Loading and selection of fish species. This class is a referential class, and it is integrated into refreport objects.

Slots

data

A 'data.frame' of species available in the database

taxa_selected

Contains the code 'tax_code' of the taxa selected by choice_c() method

Objects from the Class

Objects can be created by calls of the form new('ref_taxa', ...).
