ref_df-class R Documentation

Class 'ref_df'

Description

Representation of a fishway, contains description data of all fishways from the database along with the selected fishways (df) (integer) Objects from the Class: Objects can be created by calls of the form new('ref_df', df_selected=integer(), ouvrage=integer(), data=data.frame()).

Arguments

df_selected

Object of class 'integer' The identifier of the fishway
ouvrage

Object of class 'integer' The attached dam
data

Object of class 'data.frame' Data concerning the fishway
