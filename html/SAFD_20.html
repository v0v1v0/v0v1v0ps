XX R Documentation

Example data

Description

XX is a list of three polygonal fuzzy numbers that is used in the given examples.

Usage

data("XX")

Format

List of three polygonal fuzzy numbers.

Details

See examples

Examples

data(XX)
V<-translator(XX[[1]],50)
SS1<-vector("list",length=100)
for (j in 1:100){
 SS1[[j]]<-generator(V,)
 }
M<-Mmean(SS1,1)
head(M)
b<-bertoluzza(M,V,1/3,1) 
b
