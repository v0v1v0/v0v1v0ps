artiodactyla R Documentation

Artiodactyla Phylogeny

Description

A phylogenetic tree of Artiodactyla (even-toed ungulates).

Usage

data(artiodactyla)

Format

An object of class "phylo", from package "ape".

References

The tree is taken from Toljagic et al. (2017).

  • ToljagiÄ‡, O., Voje, K. L., Matschiner, M., Liow, L. H., & Hansen, T. F. (2017). Millions of years behind: Slow adaptation of ruminants to grasslands. Systematic Biology, (318). https://doi.org/10.1093/sysbio/syx059

Examples

data(artiodactyla)
library(ape)
plot(artiodactyla)

# Note: This tree also has regime information for each internal node in the tree:
print(artiodactyla$node.label)
