CL_OCCUPATION R Documentation

Codelist Occupation

Description

A dataset containing the Occupation codelist, abbreviated as CL_OCCUPATION. Version 2.2.

Usage

CL_OCCUPATION

Format

A data frame with 619 rows and 5 variables:

id

A statistical code for describing occupations.

name

The standard label of the code.

description

The description of the code.

name_locale

The language code of the name label.

description_locale

The language code of the description.

Details

The SDMX description: "This code list provides a list of values for describing the job or position held by an individual."
