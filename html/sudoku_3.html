hintSudoku R Documentation

Give a Hint for a Sudoku Cell

Description

Generates a text string containing a 'hint' for cell (i,j) of Sudoku grid 'z'.

Usage

  hintSudoku(z, i, j)

Arguments

z

A 9x9 numeric matrix
i

Row index
j

Column index

Value

A character string, suitable for cat.
