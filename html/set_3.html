or R Documentation

Get Union Set for Sets

Description

Get union set for sets.

Usage

or(...)

a %or% b

Arguments

...

one or more objects
a

one object
b

the other object

Value

union elements

Examples

A <- c("a","b","c")
B <- c("a","b","c","d")
C <- c("a","e","h")
or(A, B)
or(A, B, C)
A <- c("a","b","c")
B <- c("a","b","c","d")
C <- c("a","e","h")
A %or% B
A %and% B %or% C
