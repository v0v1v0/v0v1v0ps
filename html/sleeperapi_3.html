get_draft R Documentation

Gather Draft Information

Description

Given a draft ID, gather draft information for selection.

Usage

get_draft(draft_id)

Arguments

draft_id

Draft ID generated by Sleeper (integer)

Value

Returns a list containing information about the draft.
