SonarEquation R Documentation

sonar equation

Description

Returns the sonar equation EL = SL - 2PL + TS

Usage

SonarEquation(SL, PL, TS)

Arguments

SL

source level
PL

propagation loss
TS

target strength

Value

EL echo level

Author(s)

Jose Gama

References

Waite A. D., 2002 Sonar for Practising Engineers, 3rd Edition Chichester: Wiley. pp. 68.
