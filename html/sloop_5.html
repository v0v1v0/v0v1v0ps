s3_class R Documentation

Compute the S3 class of an object

Description

Compared to class(), this always returns the class vector that is used for dispatch. This is most important for objects where the class attribute has not been set.

Usage

s3_class(x)

Arguments

x

A primitive type

Examples

s3_class(NULL)

s3_class(logical())
s3_class(integer())
s3_class(numeric())
s3_class(character())

s3_class(matrix())
s3_class(matrix(1))

s3_class(array())
s3_class(array(1))
