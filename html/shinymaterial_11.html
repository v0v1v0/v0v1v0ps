material_input R Documentation

Create a shinymaterial input

Description

Build a shinymaterial input.

Usage

material_input(type, ...)

Arguments

type

String. The type of input to be created.
...

Additional arguments for the input.

Examples

##-- switch --##
material_input(
  type = "switch",
  input_id = "example_switch",
  off_label = "Off",
  on_label = "On",
  initial_value = TRUE
)
