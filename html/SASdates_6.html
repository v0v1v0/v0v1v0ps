iso8601dt R Documentation

Convert date and time to ISO8601 format

Description

Convert date and time to ISO8601 format

Usage

iso8601dt(date, time, informat)

Arguments

date

Input date
time

Input time
informat

Input date format

Value

Date and time in ISO8601 format

Examples

iso8601dt(date = "19-10-1994",time = "10:10",informat = "%Y-%m-%d %H:%M")
