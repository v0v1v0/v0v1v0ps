psychro R Documentation

Calculate psychrometric constant

Description

This function calculates the temperature and pressure dependent psychrometric constant.

Usage

psychro(tc, pa, kMa = 0.028963, kMv = 0.01802)

Arguments

tc

double, air temperature, degrees C.
pa

double, atm pressure, Pa.
kMa

double, molecular weight of dry air, kg/mol. Default: 0.028963 (Tsilingiris, 2008)
kMv

double, molecular weight of water vapor, kg/mol. Default: 0.01802 (Tsilingiris, 2008)

Value

double, Pa/K.
