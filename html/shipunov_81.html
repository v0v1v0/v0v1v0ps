Mag R Documentation

Interpreter for effect sizes

Description

Interprets R^2-related effect sizes

Usage

Mag(x, squared=TRUE)

Arguments

x

Value
squared

Is value squared?

Details

Interpreter for R^2-related effect sizes (see example).

Author(s)

Alexey Shipunov

Examples
