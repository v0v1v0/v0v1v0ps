RFactor_calc R Documentation

Determination the rainfall erosivity .

Description

The function calculates the rainfall erosivity factor.

Usage

RFactor_calc(data, erosive.precip = 10, equation = "WS")

Arguments

data

Data to be analyzed.
erosive.precip

Precipitation considered erosive (default = 10).
equation

"WS" - Wischmeier and Smith (defaul),
"BF" - Brown and Foster,
"RUSLE2" - USDA-Agriculture Research Service.

Value

result

Tabulated results.
record

Record of rainfall relative to 5, 10, 15, 30 or 60 minutes.
