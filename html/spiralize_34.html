spiral_points R Documentation

Add points to a track

Description

Add points to a track

Usage

spiral_points(
  x,
  y,
  pch = 1,
  size = unit(0.4, "char"),
  gp = gpar(),
  track_index = current_track_index()
)

Arguments

x

X-locations of the data points.
y

Y-locations of the data points.
pch

Point type.
size

Size of the points. Value should be a grid::unit() object.
gp

Graphical parameters.
track_index

Index of the track.

Value

No value is returned.

Examples

spiral_initialize()
spiral_track()
spiral_points(x = runif(1000), y = runif(1000))
