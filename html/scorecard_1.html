describe R Documentation

Variable Describe

Description

This function provides descriptive statistic for exploratory data analysis.

Usage

describe(dt)

Arguments

dt

A data frame.

Examples

library(data.table)

data("germancredit")
dat = rbind(
  setDT(germancredit),
  data.table(creditability=sample(c("good","bad"),100,replace=TRUE)),
  fill=TRUE)

eda = describe(dat)
eda
