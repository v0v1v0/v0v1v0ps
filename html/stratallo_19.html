round_oric R Documentation

Optimal Rounding under Integer Constraints

Description

Usage

round_oric(x)

Arguments

x

(numeric)
a numeric vector.

Value

An integer vector.

References

Cont, R., Heidari, M. (2014). Optimal rounding under integer constraints. doi:10.48550/arXiv.1501.00014

Examples

x <- c(4.5, 4.1, 4.9)
round_oric(x) # 4 4 5
