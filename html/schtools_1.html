schtools-package R Documentation

schtools: Schloss Lab Tools for Reproducible Microbiome Research

Description

A collection of useful functions and example code created and used by the Schloss Lab for reproducible microbiome research. Perform common tasks like read files created by mothur, tidy up your microbiome data, and format rmarkdown documents for publication. See the website for more information, documentation, and examples.
