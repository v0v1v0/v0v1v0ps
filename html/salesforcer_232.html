sf_reorder_cols R Documentation

Reorder resultset columns to prioritize sObject and Id

Description

This function accepts a tbl_df with columns rearranged.

Usage

sf_reorder_cols(df)

Arguments

df

tbl_df; the data frame to rearrange columns in

Value

tbl_df the formatted data frame

Note

This function is meant to be used internally. Only use when debugging.
