Trapeze kernel

Description

Trapeze kernel

Usage

trapeze(x, width = 0.8)

Arguments

x

a vector of real numbers.
width

a number between 0 and 1.

Value

This function computes the values of the trapeze kernel at points x.

Examples

x = seq(-2,2,length=1000)
y = trapeze(x, width=0.5)
plot(x,y)
