CL_COFOG_1999 R Documentation

Classification of the Functions of Government

Description

A dataset containing the Classification of the Functions of Government codelist, abbreviated as CL_COFOG_1999. Version 1.0.

Usage

CL_COFOG_1999

Format

A data frame with 188 rows and 5 variables:

id

A statistical code for describing government functions.

name

The standard label of the code.

description

The description of the code.

name_locale

The language code of the name label.

description_locale

The language code of the description.

Details

The SDMX description: "This code list provides a set of building blocks to be used for classifying final consumption expenditures of general government by function or purpose. It is based on the United Nations eponymous standard classification. This code list was formally adopted on 27 June 2014."
