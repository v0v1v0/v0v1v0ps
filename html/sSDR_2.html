cov.x R Documentation

Covariance matrix

Description

Covariance matrix

Usage

cov.x(X)

Arguments

X

a n x p matrix of n observations and p predictors.

Details

This function returns A p x p covariance matrix for any n x p matrix.

Value

A p x p covariance matrix.

Examples

data <- gen.data(n=100)
x.cov <- cov.x(data$X)
