quotes R Documentation

Quotes on statistics, data visualization and science

Description

A data frame with quotations. The variables are:

Usage

data(quotes)

Format

A data frame

Details

  • qid quote ID, a numeric vector

  • text text of the quote

  • source person(s) who said the quote.

  • citation citation of the quote

  • url URL of the quote

  • tags tags used for searching

  • tex TeX-style citation
