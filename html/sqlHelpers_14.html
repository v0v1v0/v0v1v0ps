sqlizeTypes R Documentation

Get the equivalent SQL data type for a given R object.

Description

Get the equivalent SQL data type for a given R object.

Usage

sqlizeTypes(x, dialect = "T-SQL")

Arguments

x

A R object.
dialect

A string, "T-SQL" or "Postgresql".

Value

A string, the equivalent SQL data type for x.

Examples

sqlizeTypes(100.1209)
