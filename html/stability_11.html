ge_effects R Documentation

Genotype by Environment Interaction Effects

Description

Calcuates Genotype by Environment Interaction Effects

Usage

ge_effects(.data, .y, .gen, .env)

## Default S3 method:
ge_effects(.data, .y, .gen, .env)

Arguments

.data

data.frame
.y

Response Variable
.gen

Genotypes Factor
.env

Environment Factor

Value

Effects
