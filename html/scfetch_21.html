PanglaoDBComposition R Documentation

All Sample Composition of PanglaoDB Datasets

Description

A dataset contains all sample composition of PanglaoDB datasets.

Usage

PanglaoDBComposition

Format

A data frame with 19,449 rows and 8 variables:

SRA

The SRA identifier

SRS

The SRS identifier

Tissue

The tissue of the sample

Protocol

The protocol used to generate this sample

Species

The species of this sample

Cluster

Seurat cluster

Cells

Cluster cell number

Cell Type

Predicted cluster cell types
