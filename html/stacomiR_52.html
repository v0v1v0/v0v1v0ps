charge,report_species-method R Documentation

charge method for report_species

Description

Verifies the content of objects when the graphical interface is used, it is not necessary to call the charge method if the choice_c method has been used

Usage

## S4 method for signature 'report_species'
charge(object, silent = FALSE)

Arguments

object

An object of class report_species-class
silent

Stops displaying the messages.

Value

report_species with slots filled by user choice

An object of class report_species-class with slots filled from values assigned in envir_stacomi environment
