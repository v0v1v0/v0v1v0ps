unmatrix R Documentation

Transform data frame with embedded matrices

Description

Transform data frame with embedded matrices

Usage

unmatrix(data, sep = "_")

Arguments

data

data frame
sep

A character string used when variable names are generated.

Value

data frame

Examples

a <- aggregate(1:6, list(rep(1:3, 2)), range)
b <- unmatrix(a)

b
