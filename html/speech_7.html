speech_url R Documentation

url vectors

Description

Allows to create a vector of url to download within a period within a legislature.

Usage

speech_url(chamber, from, to, days = NULL)

Arguments

chamber

chamber:

S

Camara de Senadores

D

Camara de Representantes (Diputados)

A

Asamblea General

C

Comision Permanente
from

character vector. Date in DD-MM-YYYY format
to

character vector. Date in DD-MM-YYYY format
days

character vector. Date in DD-MM-YYYY format.

Value

character vector

Examples

# speech_url(chamber = "D",
#            from    = "15-02-2015",
#            to      = "15-03-2015")
#
# speech_url(chamber = "D",
#            from    = "15-02-2015",
#            to      = "15-02-2015")
#
# speech_url(chamber = "D",
#            days   = "15-02-2015")
#
# speech_url(chamber = "D",
#            days    = c("12-06-2002", "14-04-2004"))
