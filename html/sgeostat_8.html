fit.trend R Documentation

Fit polynomial trend functions

Description

Fits a polynomial trend function to a point object. Similar to functions in B. Ripleys spatial library.

Usage

fit.trend(point.obj, at, np=2, plot.it=TRUE)

Arguments

point.obj

point object

at

name of dependent variable in point.obj

np

degree of polynom to be fitted

plot.it

switches generation of a contour plot

Value

beta

estimated parameters
