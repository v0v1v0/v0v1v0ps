Lower Confidence Limit

Description

The estimate of the lower bound of confidence limit using t-distribution

Usage

  LCL(y, conf.level=0.95)

Arguments

y

a vector of numerics
conf.level

confidence level

Details

It removes NA in the input vector.

Value

The estimate of the lower bound of confidence limit using t-distribution
