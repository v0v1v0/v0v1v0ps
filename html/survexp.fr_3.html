Description

Example data to illustrate the functions

Format

A data frame with 200 observations on the following 5 variables.

sex

"male" or "female"

age

age in days

entry_date

entry date in the study

status

status at follow-up time: 0 if alive, 1 if dead

futime

follow-up time in days
