  start = 0,
  end = 1000,
  step = 100,
  path = ".",

  width = 4,
  height = 3
)

data

starvz_data with trace data
start

start time
end

end time
step

between snaps

path to save files
scale

for ggsave
width

for ggsave
height

for ggsave

A ggplot object
