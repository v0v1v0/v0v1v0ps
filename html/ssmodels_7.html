IMR R Documentation

Inverse Mills Ratio

Description

Column vector of the inverse ratio of Mills

Usage

IMR(selection, data = sys.frame(sys.parent()))

Arguments

selection

Selection equation.
data

Database.

Value

Return column vector of the inverse ratio of Mills

Examples

data(MEPS2001)
attach(MEPS2001)
selectEq <- dambexp ~ age + female + educ + blhisp + totchr + ins + income
IMR(selectEq)
