enthalpy_vap R Documentation

Calculate enthalpy of vaporization

Description

This function calculates the temperature-dependent enthalpy of vaporization (latent heat of vaporization).

Usage

enthalpy_vap(tc)

Arguments

tc

double, air temperature, degrees C.

Value

double, J/kg.
