sdtm_te R Documentation

sdtm_te

Description

Trial Elements data frame - One record per planned Element

Usage

sdtm_te

Format

a data frame with 7 rows and 7 columns. See https://github.com/SafetyGraphics/safetyData/blob/main/data-raw/sdtm/README.md for more details and full data specification. Columns:

STUDYID

text - Study Identifier

DOMAIN

text - Domain Abbreviation

ETCD

text - Element Code

ELEMENT

text - Description of Element

TESTRL

text - Rule for Start of Element

TEENRL

text - Rule for End of Element

TEDUR

text - Planned Duration of Element
