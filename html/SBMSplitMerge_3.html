ARI R Documentation

Adjusted Rand Index

Description

Calculate the Adjusted Rand Index between two clusterings

Usage

ARI(z, truez)

Arguments

z

input vector
truez

reference vector

Value

Adjusted Rand Index of z against truez

Examples

ARI(c(1,1,2,2,3,3), c(2,2,1,1,3,3)) ## 1 - doesn't care for labels
ARI(c(1,1,2,2,3,3), c(1,1,1,1,2,2)) ## 0.444
