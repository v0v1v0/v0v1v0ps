Find Subobjects Of A Certain Class

Usage

.FilterObjects(object, classes.keep = c("Assay", "StdAssay", "DimReduc"))

Arguments

object

An S4 object

A vector of classes to keep

Value

A vector of object names that are of class classes.keep

Examples
