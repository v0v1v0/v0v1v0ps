derivSymbolicQspray R Documentation

Partial derivative

Description

Partial derivative of a symbolicQspray polynomial.

Usage

derivSymbolicQspray(Qspray, i, derivative = 1)

Arguments

Qspray

object of class symbolicQspray
i

integer, the dimension to differentiate with respect to, e.g. 2 to differentiate w.r.t. y
derivative

positive integer, how many times to differentiate

Value

A symbolicQspray object.
