CL_DEG_URB R Documentation

Codelist Degree of Urbanization

Description

A dataset containing the Degree of Urbanization codelist, abbreviated as CL_DEG_URB. Version 1.0.

Usage

CL_DEG_URB

Format

A data frame with 14 rows and 5 variables:

id

A statistical code for describing degrees of urbanizations.

name

The standard label of the code.

description

The description of the code.

name_locale

The language code of the name label.

description_locale

The language code of the description.

Details

The SDMX description: "This code list provides coded information about the degree to which area is urbanised."
