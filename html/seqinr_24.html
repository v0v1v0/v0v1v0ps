c2s R Documentation

conversion of a vector of chars into a string

Description

This is a simple utility function to convert a vector of chars such as c("m", "e", "r", "g", "e", "d") into a single string such as "merged".

Usage

c2s(chars = c("m", "e", "r", "g", "e", "d"))

Arguments

chars

a vector of chars

Value

a string
