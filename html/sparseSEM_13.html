Y R Documentation

Gene expression matrix

Description

Y is the M by N matrix describes the dependent attribute of M nodes in N samples

Usage

Format

The format is: num [1:30, 1:200] 3.02 1.12 -2.24 3.58 2.18 ...

Details

Gene expression data
