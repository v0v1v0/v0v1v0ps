getRMSEA.F R Documentation

getRMSEA.F

Description

Computes RMSEA from the minimum of the ML-fit-function F_min = rmsea^2 * df.

Usage

getRMSEA.F(Fmin, df, nGroups = 1)

Arguments

Fmin

minimum of the ML-fit-function
df

model degrees of freedom
nGroups

the number of groups

Value

Returns RMSEA
