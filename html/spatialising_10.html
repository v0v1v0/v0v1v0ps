Texture index

Description

Calculates texture index â€“ an average (over an array) of a product of the values of neighboring cells. The value of texture index is between 0 (fine texture), and 1 (coarse texture).

Usage

Arguments

x

SpatRaster or matrix containing two values: -1 and 1
...

Arguments for comat::get_coma()
