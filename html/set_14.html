Grep for vectors

Description

Grep for vectors

Usage

grep_or(x, patterns)

grep_and(x, patterns)

grep_not_and(x, patterns)

grep_not_or(x, patterns)

Arguments

x

vectors with one or more characters
patterns

vectors with one or more characters

Value

vectors

Examples

x=c('a12','a','b')
patterns=c('b','1')
grep_or(x,patterns)
x=c('a12','a1','b')
patterns=c('a','1')
grep_and(x,patterns)
x=c('a12','a','b')
patterns=c('a','1')
grep_not_and(x,patterns)
x=c('a12','a','b')
patterns=c('a','1')
grep_not_or(x,patterns)
