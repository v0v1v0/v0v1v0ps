siren-package R Documentation

An hybrid CFA-EFA R package for controlling acquiescence in restricted solutions

Description

Hybrid multi-stage factor analytic procedure for controlling the effects of acquiescence in multidimensional questionnaires for which a restricted factorial solution (Ferrando & Lorenzo-Seva, 2000) can be specified.

Value

acquihybrid

Performs hybrid multi-stage factor analytic procedure for controlling acquiescence in restricted solutions.

Author(s)

Pere J. Ferrando

Fabia Morales-Vives

References
