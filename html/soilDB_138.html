makeChunks R Documentation

Generate chunk labels for splitting data

Description

Generate chunk labels for splitting data

Usage

makeChunks(ids, size = 100)

Arguments

ids

vector of IDs
size

chunk (group) size

Value

A numeric vector

Examples


# split the lowercase alphabet into 2 chunks

aggregate(letters,
          by = list(makeChunks(letters, size=13)),
          FUN = paste0, collapse=",")
