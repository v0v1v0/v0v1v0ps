SyncMove-package R Documentation

Subsample Temporal Data to Synchronal Events and Compute the MCI

Description

The function syncSubsample subsamples temporal data of different entities so that the result only contains synchronal events. The function mci calculates the Movement Coordination Index (MCI, see reference on help page for function mci) e.g. of a data set created with the function syncSubsample.
