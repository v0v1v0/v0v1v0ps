The Donner Party

Description

In 1846 the Donner party became stranded while crossing the Sierra Nevada Mountains near Lake Tahoe. The data frame has the counts for male and female survivors for six age groups.

Usage

ex1918

Format

A data frame with 12 observations on the following 4 variables.

AgeCat

a numerical code corresponding to six age categories, with 1 = "15-19", 2 = "20-29", 3 = "30-39", 4 = "40-49", 5 = "50-59" and 6 = "60-69"

Sex

a factor with levels "female" and "male"

Lived

number that lived

Died

number that died

Source

Ramsey, F.L. and Schafer, D.W. (2013). The Statistical Sleuth: A Course in Methods of Data Analysis (3rd ed), Cengage Learning.
