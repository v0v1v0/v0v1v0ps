Get TMB parameter list

Description

Get TMB parameter list

Usage

get_pars(object)

Arguments

object

Fit from sdmTMB()

Value

A named list of parameter values

Examples

fit <- sdmTMB(present ~ 1, data = pcod_2011, family = binomial(), spatial = "off")
pars <- get_pars(fit)
names(pars)
