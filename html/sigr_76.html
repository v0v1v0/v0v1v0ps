wrapChiSqTest.summary.glm R Documentation

Format ChiSqTest from model summary.

Description

Format ChiSqTest from model summary.

Usage

## S3 method for class 'summary.glm'
wrapChiSqTest(x, ...)

Arguments

x

summary(glm(family=binomial)) object.
...

extra arguments (not used)

Value

wrapped test

Examples


d <- data.frame(x=c(1,2,3,4,5,6,7,7),
      y=c(TRUE,FALSE,FALSE,FALSE,TRUE,TRUE,TRUE,FALSE))
model <- glm(y~x,data=d,family=binomial)
sum <- summary(model)
render(wrapChiSqTest(sum),pLargeCutoff=1,format='ascii')
