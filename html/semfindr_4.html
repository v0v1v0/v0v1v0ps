cfa_dat2 R Documentation

Sample Data: A CFA Model with an Influential Case

Description

A six-variable dataset with 100 cases, with one influential case.

Usage

cfa_dat2

Format

A data frame with 100 rows and 7 variables:

case_id

Case ID. Character.

x1

Indicator. Numeric.

x2

Indicator. Numeric.

x3

Indicator. Numeric.

x4

Indicator. Numeric.

x5

Indicator. Numeric.

x6

Indicator. Numeric.

Examples

library(lavaan)
data(cfa_dat2)
mod <-
"
f1 =~  x1 + x2 + x3
f2 =~  x4 + x5 + x6
"
fit <- cfa(mod, cfa_dat2)
summary(fit)
inf_out <- influence_stat(fit)
gcd_plot(inf_out)
