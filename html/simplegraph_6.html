Edges of a graph

Description

Edges of a graph

Usage

edges(graph)

Arguments

graph

The graph

Value

Data frame of edge data and metadata. The tail and head vertices are in the fist two columns. The rest of the columns are metadata.

See Also

Other simple queries: adjacent_vertices(), order(), vertices()

Examples

bridges <- graph(list(
  "Altstadt-Loebenicht" = c(
    "Kneiphof",
    "Kneiphof",
    "Lomse"
  ),
  "Kneiphof" = c(
    "Altstadt-Loebenicht",
    "Altstadt-Loebenicht",
    "Vorstadt-Haberberg",
    "Vorstadt-Haberberg",
    "Lomse"
  ),
  "Vorstadt-Haberberg" = c(
    "Kneiphof",
    "Kneiphof",
    "Lomse"
  ),
  "Lomse" = c(
    "Altstadt-Loebenicht",
    "Kneiphof",
    "Vorstadt-Haberberg"
  )
))
edges(bridges)
