Arguments

object

An object
x, y

Range to crop x/y limits to; if NULL, uses full range of x/y
coords

Coordinate system to execute crop; choose from:

  • “plot”: Coordinates as shown when plotting

  • “tissue”: Coordinates from GetTissueCoordinates
