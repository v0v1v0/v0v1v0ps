Usage

yearlyPlot(
  year = 2022,
  lat,
  lon,
  limits = c(0, 2),
  plot = NULL,
  method = "plotrix",
  legend = F
)

Arguments

year

Year to plot (allows for leap years).
lat

Numeric latitude.
lon

Numeric longitude.
limits

Plotting limits of the daylight regions, default to c(1,2)
plot

Character vector of components to plot
method

Plotting library to use
legend

Whether to show a legend
