BasicSonarEquation R Documentation

basic sonar equation

Description

Returns the basic sonar equation SE = S - N + DT

Usage

BasicSonarEquation(S, N, DT)

Arguments

S

signal
N

noise
DT

detection threshold

Value

SE signal excess (dB)

Author(s)

Jose Gama

References

Waite A. D., 2002 Sonar for Practising Engineers, 3rd Edition Chichester: Wiley. pp. 120.
