random_distpars R Documentation

Create random distribution parameter values

Description

random_distpars generates random distribution parameter values

Usage

random_distpars(d, cond_dist)

Arguments

cond_dist

specifies the conditional distribution of the model as "Gaussian", "Student", or "ind_Student", where the latest is the Student's t distribution with independent components.

Value

Returns a numeric vector ...

If cond_dist == "Gaussian":

of length zero.

If cond_dist == "Student":

of length one containing a df param strictly larger than two.

If cond_dist == "ind_Student":

of length d containing a df params strictly larger than two.
