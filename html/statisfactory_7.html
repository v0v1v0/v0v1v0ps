euclid R Documentation

Euclidean distance

Description

Euclidian distance in one or more dimensions.

Usage

euclid(a, b, na.rm = FALSE)

Arguments

a

Numeric vector.
b

Numeric vector of same length as a.
na.rm

Logical. If TRUE, calculation ignores NA's in a and/or b.

Value

Numeric.

Examples


euclid(0, 5)
euclid(c(0, 0), c(1, 1))
euclid(c(0, 0, 0), c(1, 1, 1))
