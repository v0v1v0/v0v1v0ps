normc R Documentation

default normalization function for matrix

Description

normalize columns of a matrix.

Usage

normc(X)

Arguments

X

the input martix

Value

Y: the output matrix

Examples

library(mvtnorm)
n = 350
p = 100
Sigma = matrix(0, p, p)
X <- rmvnorm(n,matrix(0, p, 1), Sigma)
X <- normc(X)
