BreesPass R Documentation

Drew Brees Passing Statistics (2016)

Description

Passing statistics for football quarterback Drew Brees in 2016

Format

A data frame with 16 observations on the following 5 variables.

Game

Game number (1 is the first game of the regular season)

Opponent

Opponent abbreviation

Completed

Number of completed passes

Attempts

Pass attempts

Yards

Passing yards

Details

Drew Brees was the quarterback for the NFL's New Orleans Saints football team in 2016. This dataset shows some of his passing statistics for each of the 16 regular season games.

Source

http://www.espn.com/nfl/player/gamelog/_/id/2580/year/2016
