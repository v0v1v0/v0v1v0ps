is.sr R Documentation

Is this in the "sr" class?

Description

Checks if an object is in the class 'sr'

Usage

is.sr(x)

Arguments

x

an object of some kind.

Details

To satisfy the minimum requirements of an S3 class.

Value

a boolean.
