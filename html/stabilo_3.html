sttkeccentr R Documentation

Estimates the Eccentricity of a Statokinesigram.

Description

Computes the eccentricity of the confidence ellipse of a given statokinesigram.

Usage

sttkeccentr(dados)

Arguments

dados

data frame with two columns "x" and "y"

Details

'dados' is a data frame containing two columns named "x" and "y". The pairs (x, y) are the coordinates of the center of pressure acquired in a period of time.

Value

The eccentricity of the given statokinesigram eccentr.

Author(s)

Jose Magalhaes de Oliveira

See Also

sttkangle, sttkarea

Examples

x <- c(1,3,7,5,9,4,3,6,8,2,8,9,4,5,7,3,4,7,9,3,2,5,3,4,8,2,9,7,4,2)
y <- c(6,3,9,1,3,7,4,9,6,1,7,3,9,7,2,6,3,4,8,1,9,3,6,8,1,6,2,9,8,3)

COP <- data.frame(x, y)

COPeccentr <- sttkeccentr(COP)
