extractCorr R Documentation

Extracting correlations from a covariance matrix

Description

Extracting correlations from a covariance matrix

Usage

extractCorr(mat)

Arguments

mat

A covariance matrix.

Value

The correlation matrix embedded in mat.

Examples

# 2 dimensional case
d <- 2
tmp <- matrix(rnorm(d^2), d, d)
mcov <- tcrossprod(tmp, tmp)

# Covariance matrix
mcov

# Correlation matrix
extractCorr(mcov)
