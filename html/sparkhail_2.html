hail_context R Documentation

Create Hail Context

Description

Import and initialize Hail using a spark connection.

Usage

hail_context(sc)

Arguments

sc

Spark connection.

Value

hailContext

Examples

library(sparklyr)

sc <- spark_connect(master = "spark://HOST:PORT", config = hail_config())
connection_is_open(sc)
hail_context(sc)
spark_disconnect(sc)
