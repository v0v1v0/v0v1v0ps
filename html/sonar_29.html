PLsphericalSpreadingLaw R Documentation

PL to range r spherical spreading law in logarithmic form

Description

Returns the PL to range r spherical spreading law in logarithmic form

Usage

PLsphericalSpreadingLaw(r)

Arguments

r

radius (meters)

Value

Propagation loss (PL) (dB)

Author(s)

Jose Gama

References

Examples

PLsphericalSpreadingLaw( 1000 )
