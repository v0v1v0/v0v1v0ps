printSudoku R Documentation

Print a Sudoku Grid to the Terminal.

Description

Prints a Sudoku grid (a 9x9 matrix) to the terminal.

Usage

  printSudoku(z)

Arguments

z

A 9x9 numeric matrix, with '0' representing a blank cell.

Value

None; used for side effect.
