matpower R Documentation

Power of a matrix

Description

Power of a matrix

Usage

matpower(X, alpha)

Arguments

X

A p x p square matrix.
alpha

A scaler determining the order of the power.

Details

This function calculates the power of a square matrix.

Value

A p x p square matrix.

Examples

data <- gen.data(n=100)
cov.squared <- matpower(cov.x(data$X), 2)
