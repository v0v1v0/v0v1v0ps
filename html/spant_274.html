resample_voi R Documentation

Resample a VOI to match a target image space using nearest-neighbour interpolation.

Description

Resample a VOI to match a target image space using nearest-neighbour interpolation.

Usage

resample_voi(voi, mri)

Arguments

voi

volume data as a nifti object.
mri

image data as a nifti object.

Value

volume data as a nifti object.
