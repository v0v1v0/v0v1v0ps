backquote R Documentation

Wrap in backquotes

Description

Wraps strings in backquotes.

Usage

backquote(x)

Arguments

x

A character vector.

Value

A character vector.

Note

Existing backquote characters are escaped with a backslash.

See Also

sQuote

Examples

## Not run: 
backquote(c("foo bar", "a`b`c"))

## End(Not run)
