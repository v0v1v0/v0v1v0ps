getF R Documentation

getF

Description

Computes the minimum of the ML-fit-function from known fit indices.

Usage

getF(
  effect,
  effect.measure,
  df = NULL,
  p = NULL,
  SigmaHat = NULL,
  Sigma = NULL,
  muHat = NULL,
  mu = NULL,
  fittingFunction = "ML"
)

Arguments

effect

magnitude of effect
effect.measure

measure of effect, one of 'fmin', 'rmsea', 'agfi', 'gfi', 'mc'
df

model degrees of freedom
p

number of observed variables
SigmaHat

model implied covariance matrix
Sigma

observed (or population) covariance matrix
muHat

model implied mean vector
mu

observed (or population) mean vector
fittingFunction

one of ML (default), WLS, DWLS, ULS

Value

Returns Fmin
