renameTXT R Documentation

renameTXT

Description

rename txt file (eg Description line of fasta file) according to first token (eg accession number)

Usage

renameTXT(txt_file, name_file, sep = "_", split = TRUE)

Arguments

txt_file

txt file
name_file

name file
sep

separator
split

logical, split result or not

Value

None
