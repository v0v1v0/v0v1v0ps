codebooks R Documentation

Available codelists by concept

Description

A dataset containing the available codelists in the StatCodelists package. Currently it contains only SDMX harmonized concepts and their codelists.

Usage

codebooks

Format

A data frame with 21 rows and 3 variables:

concept

A statistical concept, such as Age, Sex, Currency.

codebook

The name of the codebook, for example, CL_AGE.

authority

The standardizing authority, in this case, SDMX
