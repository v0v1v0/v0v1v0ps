Read.tri.nts R Documentation

Read 'NTSYSpc' files

Description

Read a lower triangular matrix

Usage

Arguments

file

File to read
...

Arguments to 'scan()'

Details

Reads a lower triangular matrix which at least in my practice, typically come from 'NTSYSpc' program.
