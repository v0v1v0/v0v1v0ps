VisiumV2-class R Documentation

The VisiumV2 class

Description

The VisiumV2 class represents spatial information from the 10X Genomics Visium HD platform - it can also accomodate data from the standard Visium platform

Slots

image

A three-dimensional array with PNG image data, see readPNG for more details

scale.factors

An object of class scalefactors; see scalefactors for more information
