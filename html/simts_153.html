is.whole R Documentation

Integer Check

Description

Checks whether the submitted value is an integer

Usage

is.whole(x)

Arguments

x

A numeric value.

Value

A boolean value indicating whether the value is an integer or not.

Author(s)

James Balamuta

Examples

is.whole(2.3)
is.whole(4)
is.whole(c(1,2,3))
is.whole(c(.4,.5,.6))
is.whole(c(7,.8,9))
