ClassKey R Documentation

Generate a Class Key

Description

Generate class keys for S4 classes. A class key follows the following structure: “package:class”

Usage

ClassKey(class, package = NULL)

Arguments

class

Class name
package

Optional name of package; by default, will search namespaces of loaded packages to determine the providing package

Value

The class key (“package:class”)

See Also

s4list

Examples

ClassKey("Seurat")
