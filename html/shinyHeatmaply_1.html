html2tagList R Documentation

html2tagList

Description

convert raw html to tagList

Usage

html2tagList(x)

Arguments

x

character vector of html

Examples

x<-'<h1>Title</h1>
   <h2>Header text</h2>
   <p>Text here</p>
   <h1>Title</h1>
   <h2>Header text</h2>
   <p>Text here</p>'
   
 html2tagList(x)
