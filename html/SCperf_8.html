SCperf R Documentation

Deprecated functions in the SCperf package

Description

The functions listed below are provided for compatibility with older version of the SCperf package. They have been deprecated and will be defunct in the next version of this package. Help pages for deprecated functions are available at help("-deprecated")

Usage

SCperf(phi, theta, L = L, SL = 0.95)

SSL(method = c("MMSE", "SMA", "ES"), phi, L, p, alpha, SL)

bullwhip(method = c("MMSE", "SMA", "ES"), phi, L, p, alpha)

Details

The SCperf function has been deprecated and will be made defunct; use the bullwhipgame package.

The SSL function has been deprecated and will be made defunct; use the bullwhipgame package.

The bullwhip function has been deprecated and will be made defunct; use the bullwhipgame package.
