fas2phy R Documentation

fas2phy

Description

convert fasta (aligned sequences) to phylip format

Usage

fas2phy(fasfile, outfile = "out.phy", type = "sequential")

Arguments

fasfile

aligned sequences in fasta format
outfile

output file
type

one of interleaved and sequential

Value

None
