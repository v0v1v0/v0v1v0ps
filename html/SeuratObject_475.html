set-if-null R Documentation

Set If or If Not NULL

Description

Set a default value depending on if an object is NULL

Usage

x %||% y

x %iff% y

Arguments

x

An object to test
y

A default value

Value

For %||%: y if x is NULL; otherwise x

For %iff%: y if x is not NULL; otherwise x

Author(s)

For %||%: rlang developers

See Also

rlang::%||%

Examples

# Set if NULL
1 %||% 2
NULL %||% 2

# Set if *not* NULL
1 %iff% 2
NULL %iff% 2
