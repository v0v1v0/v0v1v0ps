PanglaoDBMeta R Documentation

All Sample Metadata of PanglaoDB Datasets

Description

A dataset contains all sample metadata and cell types of PanglaoDB datasets.

Usage

PanglaoDBMeta

Format

A data frame with 1,368 rows and 7 variables:

SRA

The SRA identifier

SRS

The SRS identifier

Tissue

The tissue of the sample

Protocol

The protocol used to generate this sample

Species

The species of this sample

Cells

Total cell number of this sample

CellType

Predicted cell types, separated by comma
