getK R Documentation

Estimate the number of common factors K

Description

Estimate the number of common factors K

Usage

getK(y, X, Kmax = 12)

Arguments

y

Response, T by 1 vector
X

Predictors, p by T matrix
Kmax

A prescribed upper bound that possibly increases with p and T (default = 12)

Value

Estimate of K
