orthnormal R Documentation

Gram-Schmidt orthonormalization

Description

Gram-Schmidt orthonormalization

Usage

orthnormal(X)

Arguments

X

a n x p matrix of n observations and p predictors.

Details

This function orthonormalizes any n x p matrix.

Value

A n x p matrix of n observations and p predictors.

Examples

data <- gen.data(n=100)
x.orth <- orthnormal(data$X)
