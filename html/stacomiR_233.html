ref_dc-class R Documentation

Class 'ref_dc'

Description

Description of a control device.

Slots

dc_selected

Object of class 'integer', The selected device

ouvrage

Object of class 'integer', the attached dam

station

Object of class 'character', the attached migration monitoring station, this is necessary to join the table of escapements calculated at the station level.

data

Object of class 'data.frame' data pertaining to the control device

Objects from the Class

Objects can be created by calls of the form new('ref_dc', dc_selected=integer(), ouvrage=integer(), data=data.frame()).
