H.omega.sincos R Documentation

Bases Transformation

Description

Create H as sine-cosine bases expansion over R^1

Usage

H.omega.sincos(x, u)

Arguments

x

A vector of locations of length n.

u

A vector of frequencies of length d/2.

Value

An n x d matrix.

See Also

H.omega.cos.2D

Examples

x <- I(0:10) / 10
u <- I(1:4) * pi
H.omega.sincos(x, u)
