Conf Matrix

Description

Calculates a cross-tabulation of observed and predicted classes with associated statistics.

Usage

confmatrix(actuals, preds)

Arguments

actuals

a numeric vector
preds

a numeric vector

Details

confmatrix

Value

A table same as caret::ConfusionMatrix

Author(s)

Examples

confmatrix(c(1,1,1,0), c(1,1,0,0))
