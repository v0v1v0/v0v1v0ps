dot R Documentation

Dot product similarity in vectors

Description

Computes the dot product between two given vectors.

Usage

dot(a, b, norm = TRUE)

Arguments

a

numeric vector
b

numeric vector
norm

logical, compute normalised dot product, default=True

Value

numeric vector containing sdot product score

Examples

a <- runif(5)
b <- runif(5)
s <- dot(a, b)
