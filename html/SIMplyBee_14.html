Combine a NULL and AlphaSimR population

Description

This combine c() method is a hack to combine NULL and an AlphaSimR population object c(NULL, pop) (c(pop, NULL) works already with AlphaSimR package code).

Usage

c(x, ...)

NULL or Pop-class
