eda.stats R Documentation

Summary statistics of a numeric array.

Description

Generates summary statistics of mean, median, skew, kurtosis, min, max and quartiles.

Usage

eda.stats(x)

Arguments

x

numeric array

Details

requires timeDate library

Value

Named list with the following items:

summary

min, 1st quartile, median, mean, 3rd quartile, max
skew

skewness
kurt

kurtosis

Author(s)

Garry Grabow

See Also

eda.shape, summary,skewness, kurtosis

Examples

require(timeDate)
# summary statistics for random normal data
# mean of 1. sd = 0.3
ndata<-rnorm(25,1,.3)
eda.stats(ndata)
#summary statistics for right-skewed data
#mean of 1, sd=1
rdata<-rexp(25,rate=1)
eda.stats(rdata)
