Trimmed Mean

Description

Trimmed mean wrapping mean function.

Usage

  trimmedMean(y, Trim=0.05)

Arguments

y

a vector of numerics
Trim

trimming proportion. Default is 0.05

Details

It removes NA in the input vector.

Value

The value of trimmed mean
