ref_year-class R Documentation

Year reference class

Description

Class used to select one or several years

Slots

data

A data.frame with the list of possible years selected as numerics

year_selected

A numeric vector

Objects from the Class

Objects can be created by calls of the form new("ref_year", data=data.frame(), year_selected=numeric()).
