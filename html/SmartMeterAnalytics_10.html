encode_p_val_stars R Documentation

Encodes p-values with a star rating according to the Significance code:

Description

'.' for p-value < 0.1, '*' for < 0.05, '**' for < 0.01, '***' for < 0.001

Usage

encode_p_val_stars(pval)

Arguments

pval

the p-value

Value

character with the encoding
