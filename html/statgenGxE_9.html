createVarComp R Documentation

S3 class varComp

Description

Function for creating objects of S3 class varComp.
print, summary, and plot methods are available.

Usage

createVarComp(
  fitMod,
  modDat,
  trait,
  nestingFactor,
  useLocYear,
  useRegionLocYear,
  fullRandVC,
  aovFullFixedMod,
  engine,
  diagTabs
)

Arguments

fitMod

A fitted variance components model.
modDat

A data.frame containing the data used in fitting the model.
