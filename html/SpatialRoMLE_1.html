IPS_coefs R Documentation

Initial coefficients of individual pension system data

Description

Initial coefficients of individual pension system data were obtained by MLE.

Usage

IPS_coefs

Format

A list with 10 values, which are:

(Intercept)

intercept

Labor_Rate

labor rate

Unemployment_Rate

unemployment rate

Sex_Ratio

sex ratio

Urbanization_Rate

urbanization rate

Deposit_Rate

deposit rate

Illiteracy_Rate

illiteracy rate

HDI

human development index

lambda

spatial autocorrelation parameter

s2

variance
