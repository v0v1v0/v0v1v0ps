SINRELEF.LD_GUI R Documentation

Score Information, reliability, and relative efficiency in measures that contain locally-dependent items, using a Graphical User Interface.

Description

A Graphical User Interface (GUI) caller for the SINRELEF.LD main function. The GUI contains all the required inputs for the execution of SINRELEF.LD, as well as the output display on console.

For more info about the inputs required, or the output information, please SINRELEF.LD.

Usage

SINRELEF.LD_GUI()
