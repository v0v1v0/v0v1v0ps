Evaluation.KendallTau R Documentation

Calculates the Kendall Tau correlation between two ranks

Description

Calculates the Kendall Tau correlation between two ranks

Usage

Evaluation.KendallTau(rank1, rank2)

Arguments

rank1

two rankings. Order does not matter
rank2

two rankings. Order does not matter

Value

The Kendall Tau correlation

Examples

rank1 <- scramble(1:10)
rank2 <- scramble(1:10)
Evaluation.KendallTau(rank1, rank2)
