createOptions R Documentation

Generates a list of option

Description

For f7Radio and f7CheckboxGroup

Usage

createOptions(
  inputId,
  choices,
  selected,
  position = "start",
  has_media_list = FALSE,
  type
)

Arguments

inputId

Radio input id.
choices

List of choices.
selected

Selected value
position

Check mark position.
has_media_list

For custom choices.
type

Choose either "checkbox" or "radio"
