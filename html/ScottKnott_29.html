summary R Documentation

Summary Method for SK Objects

Description

Returns (and prints) a summary list for SK objects.

Usage

## S3 method for class 'SK'
summary(object,
 ...)

Arguments

object

A given object of the class SK.
...

Potential further arguments (required by generic).
