prune_sg R Documentation

Prune a graph

Description

Prune a graph

Usage

prune_sg(g, level = 1, verbose = FALSE)

Arguments

g

sg object
level

pruning level
verbose

verbosity

Details

Remove edges from a graph by their path connectivity.

Examples

x <- matrix(runif(50*2), ncol=2)
g <- spatgraph(x, "MST")
gp <- prune_sg(g, level = 2)
plot(g, x, lty=2)
plot(gp, x, add=TRUE, col=2)
