equiv R Documentation

Equiv

Description

Determine if two sparta objects are equivalent

Usage

equiv(x, y)

## S3 method for class 'sparta'
equiv(x, y)

Arguments

x

sparta object
y

sparta object

Value

Logical. TRUE if x and y are equivalent

Examples


x <- array(
  c(1,0,0,2,3,4,0,0),
  dim = c(2,2,2),
  dimnames = list(
    a = c("a1", "a2"),
    b = c("b1", "b2"),
    c = c("c1", "c2")
  )
)

y <- array(
  c(2,0,0,2,3,4,0,0),
  dim = c(2,2,2),
  dimnames = list(
    a = c("a1", "a2"),
    b = c("b1", "b2"),
    c = c("c1", "c2")
  )
)

sx <- as_sparta(x)
sy <- as_sparta(y)

equiv(sx, sy)
equiv(sx, sx)
