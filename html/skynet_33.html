skynet-package R Documentation

skynet: Network analysis for BTS Data

Description

Creates networks from the BTS/Transtats data

Details

Given the DB1BCoupon and DB1BTicket, or the T-100 csv's exported this package allows creating data frames and subsequent igraph graphs.
