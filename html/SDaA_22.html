otters R Documentation

Otters Data

Description

Data on number of holts (dens) in Shetland, United Kingdom used in Kruuk et al. (1989). (Data courtesy of Hans Kruuk).

Usage

otters

Format

Data frame with the following three variables:

section

coastline section

habitat

type of habitat (stratum)

holts

number of holts

Source

Kruuk, H.A. et al. (1989). An estimate of numbers and habitat preferences of otters Lutra lutra in Shetland, UK., Biological Conservation, 49: 241â€“254.
