loadCovs R Documentation

loadCovs

Description

Load Hosts covariates

Usage

loadCovs(lhost, dfCovs, id = "ID", colCovs)

Arguments

lhost

a HostSet
dfCovs

a data.frame with host ID in rows and covariates in columns
id

colname for host ID
colCovs

colnames of covariates columns

Value

lhost updated with covariates
