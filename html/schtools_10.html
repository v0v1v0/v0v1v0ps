is_nondesc R Documentation

Check whether all elements given are sorted in non-descending order

Description

Check whether all elements given are sorted in non-descending order

Usage

is_nondesc(...)

Arguments

...

anything!

Value

TRUE if the elements are sorted in non-descending order, otherwise FALSE

Author(s)

Kelly Sovacool sovacool@umich.edu

Examples


is_nondesc(1, 2, 3)
is_nondesc(c(1, 2), 3)
is_nondesc(6, 4, 1)
is_nondesc("a", "b", "c")
is_nondesc(c("z", "y"))
