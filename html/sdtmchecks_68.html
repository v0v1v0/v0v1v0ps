check_ex_visit R Documentation

Check for missing EX.VISIT

Description

This check looks missing EX.VISIT values when EX.EXOCCUR=Y (or EX.EXOCCUR doesn't exist)

Usage

check_ex_visit(EX)

Arguments

EX

Exposure SDTM dataset with variables USUBJID,EXTRT,EXSTDTC,VISIT, and optional variable EXOCCUR

Value

boolean value if check failed or passed with 'msg' attribute if the test failed

Author(s)

Jen Chen

Examples


 EX <- data.frame(
 USUBJID = 1:3,
 EXTRT = 1:3,
 EXSTDTC = 1:3,
 EXOCCUR = "Y",
 VISIT = NA
)

check_ex_visit(EX)

EX$EXOCCUR=NULL

check_ex_visit(EX)

EX$VISIT=NULL

check_ex_visit(EX)#
