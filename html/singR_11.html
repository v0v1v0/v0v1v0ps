Generate initialization from specific space

Description

Generate initialization from specific space

Usage

gen.inits(p, d, runs, orth.method = c("svd", "givens"))

Arguments

p

p*p orthodox matrix
d

p*d orthodox matrix
runs

the number of orthodox matrix
orth.method

orthodox method

Value

a list of initialization of mixing matrices.

Examples

gen.inits(2,3,3,'svd')
