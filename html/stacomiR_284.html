r_mig R Documentation

Video counting of Marine lamprey (Petromyzon marinus) in 2012 in the Vilaine (France)

Description

This dataset corresponds to the data collected at the vertical slot fishway in 2012, video recording marine lamprey migration

Usage

r_mig

Format

An object of class report_mig with 8 slots:

dc

the ref_dc object with 4 slots filled with data corresponding to the iav postgres schema

taxa

the ref_taxa the taxa selected

stage

the ref_stage the stage selected

timestep

the ref_timestep_daily calculated for all 2015

data

A dataframe with 10304 rows and 11 variables

ope_identifiant

operation id

lot_identifiant

sample id

lot_identifiant

sample id

ope_dic_identifiant

dc id

lot_tax_code

species id

lot_std_code

stage id

value

the value

type_de_quantite

either effectif (number) or poids (weights)

lot_dev_code

destination of the fishes

lot_methode_obtention

method of data collection, measured, calculated...

coef_conversion

A data frame with 0 observations : no quantity are reported for video recording of mullets, only numbers

time.sequence

A time sequence generated for the report, used internally
