correlation_groups R Documentation

Groups layers based on the Pearson correlation

Description

correlation_groups returns groups of layer codes such as each layer from one group has an absolute Pearson product-moment correlation coefficient (Pearson's r) that is smaller than the maximum_correlation (default 0.7) with each variable in any other group. The correlation values of quadratic layers are used for creating the groups but only non quadratic layer codes are returned.

Usage

correlation_groups(layers_correlation, max_correlation=0.7)

Arguments

layers_correlation

matrix or dataframe. A square matrix with the layers correlations you want to group.
max_correlation

number. The maximum correlation 2 layers may have before they are put in the same correlation group.

Value

A list of vectors with each vector containing the layer codes of one correlation group.

Examples
