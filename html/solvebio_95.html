Task.retrieve R Documentation

Task.retrieve

Description

Retrieves the metadata about a specific task on SolveBio.

Usage

Task.retrieve(id, env = .solveEnv)

Arguments

id

The ID of a task.
env

(optional) Custom client environment.

References

https://docs.solvebio.com/

Examples

## Not run: 
Task.retrieve("1234567890")

## End(Not run)
