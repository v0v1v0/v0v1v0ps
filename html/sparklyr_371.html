sdf_sort R Documentation

Sort a Spark DataFrame

Description

Sort a Spark DataFrame by one or more columns, with each column sorted in ascending order.

Usage

sdf_sort(x, columns)

Arguments

x

An object coercable to a Spark DataFrame.
columns

The column(s) to sort by.
