Tag R Documentation

React component builder

Description

Create a React component. This is similar to React.

Usage

Tag

Format

An object of class ReactTagBuilder of length 0.

Examples

Tag$Box(
  bg = "tomato",
  Tag$ButtonGroup(
    spacing = "4",
    Tag$Button(
      "I'm a button"
    ),
    Tag$Button(
      "I'm another button"
    )
  )
)
