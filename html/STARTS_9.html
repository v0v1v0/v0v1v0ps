data.starts R Documentation

Datasets in the STARTS Package

Description

Some datasets for illustration used in the examples of the STARTS package.

Usage

data(data.starts01a)
data(data.starts01b)
data(data.starts02)
data(data.starts03a)
data(data.starts03b)
data(data.starts03c)
