CL_SEASONAL_ADJUST R Documentation

Codelist Seasonal Adjustment

Description

Codelist Seasonal Adjustment, abbreviated CL_SEASONAL_ADJUST Version 1.0

Usage

CL_SEASONAL_ADJUST

Format

A data frame with 11 rows and 5 variables:

id

A statistical code for describing seasonal adjustments.

name

The standard label of the code.

description

The description of the code.

name_locale

The language code of the name label.

description_locale

The language code of the description.

Details

The SDMX description: 'Seasonal adjustment is a statistical technique used to remove the effects of seasonal and calendar influences operating on a data series. Version 1.0 of the code list was formally adopted by the SDMX Secretariat on 4 July 2014. '
