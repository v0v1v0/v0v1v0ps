Visualization.MultiType R Documentation

Multitype Random Utility visualizer

Description

Multitype Random Utility visualizer

Usage

Visualization.MultiType(multitype.output, min, max, names, ncol)

Arguments

multitype.output

output from a multitype fitter
min

left boundary of graphed x-axis
max

right boundary of graphed x-axis
names

names of alternatives
ncol

number of columns in final output

Value

none

Examples

library(ggplot2)
library(gridExtra)
Data.Tiny <- matrix(c(1, 2, 3, 3, 2, 1, 1, 2, 3), ncol = 3, byrow = TRUE)
multitype.output <- Estimation.RUM.MultiType.MLE(Data.Tiny, iter = 1, dist = "norm", ratio = .5)
