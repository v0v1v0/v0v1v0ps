check_survey_metadata R Documentation

Check survey metadata

Description

Returns title/description HTML tags as needed.

Usage

check_survey_metadata(survey_description, survey_title)

Arguments

survey_description

The survey's description from surveyOutput
survey_title

The survey's title from surveyOutput

Value

Returns error messages if required paramters are not supplied, otherwise it returns the appropriate code for survey titles and description for use in surveyOutput.
