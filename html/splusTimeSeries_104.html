seriesLength R Documentation

Length of a timeSeries

Description

Returns the length of a timeSeries; that is, it returns the number of positions in the timeSeries.

Usage

seriesLength(x)

Arguments

x

an object of class timeSeries.

Value

returns the length of the timeSeries.

Note

This function is distinguished from the length function, which returns the number of series in the timeSeries object.

See Also

timeSeries class.

Examples

x <- timeSeries(data=data.frame(x=1:10, y=11:20), from="7/4/2000", by="bizdays")
seriesLength(x)
length(x)
