combine.zones R Documentation

Combine distinct zones

Description

combine.zones combines the elements of z1 and z2 into a single list, returning only the unique zones.

Usage

combine.zones(z1, z2)

Arguments

z1

A list of zones
z2

A list of zones

Value

A list of distinct zones

Examples

z1 <- list(1:2, 1:3)
z2 <- list(2:1, 1:4)
combine.zones(z1, z2)
