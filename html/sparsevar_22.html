plotIRF R Documentation

IRF plot

Description

Plot a IRF object

Usage

plotIRF(irf, eb, i, j, type, bands)

Arguments

irf

the irf object to plot
eb

the errorbands to plot
i

the first index
j

the second index
type

type = "irf" or type = "oirf"
bands

"quantiles" or "sd"

Value

An image plot relative to the impulse response function.
