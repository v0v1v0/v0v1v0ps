root.matrix.crossprod R Documentation

Root of X^TX

Description

Computes the root of the Gramian X^TX.

Usage

root.matrix.crossprod(X)

Arguments

X

a matrix

Value

a symmetric matrix V where V^2 equals X^TX

Examples

set.seed(1)
n <- 100
p <- 3
X <- matrix(rnorm(n*p),nrow=n, ncol=p)
test <- root.matrix.crossprod(X)
## control results
t(X) %*% X
test %*% test
