check_ts_aedict R Documentation

Check for missing MedDRA version in TS

Description

This check looks for missing MedDRA version; if it's present, also checking it's the current version

Usage

check_ts_aedict(TS)

Arguments

TS

Trial Summary SDTM dataset with variables TSPARMCD and TSVAL

Value

boolean value if check failed or passed with 'msg' attribute if the test failed

Author(s)

Vira Vrakina, Antony Howard (HackR 2021 Team Pentraxin1)

Examples


TS1 <- data.frame(
 STUDYID = 1,
 TSPARMCD = "AEDICT",
 TSVAL = "MedDRA 22.0",
 TSVAL2 = ""
)

TS2 <- data.frame(
 STUDYID = 2,
 TSPARMCD = "AEDICT",
 TSVAL = "",
 TSVAL1 = "meddra v22.0"
)

TS3 <- data.frame(
 STUDYID = 3,
 TSPARMCD = "AEDICT",
 TSVAL = ""
)

TS4 <-data.frame(
 STUDYID = 4,
 TSPARMCD = "CMDICT",
 TSVAL = ""
)

TS5 <- data.frame(
    STUDYID = 1,
    TSPARMCD = "AEDICT",
    TSVAL = "meddra 24.0",
    TSVAL2 = ""
)

TS6 <- data.frame(
    STUDYID = 1,
    TSPARMCD = "AEDICT",
    TSVAL = "    meddra    23.0   ",
    TSVAL2 = ""
)

check_ts_aedict(TS1)
check_ts_aedict(TS2)
check_ts_aedict(TS3)
check_ts_aedict(TS4)
check_ts_aedict(TS5)
check_ts_aedict(TS6)
check_ts_aedict(rbind(TS1,TS1))
