zero.pad R Documentation

Zero Pad Integers

Description

Return character representation of integers that are zero-padded to the left so all are the same length.

Usage

zero.pad(x)

Arguments

x

a vector of integers.

Author(s)

Eric Archer eric.archer@noaa.gov

Examples

x <- c(0, 1, 3, 4, 10) 
zero.pad(x)
x <- c(x, 11, 12, 100, 1000)
zero.pad(x)
