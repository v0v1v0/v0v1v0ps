Predecessors and successors

Description

Predecessors and successors

Usage

predecessors(graph)

successors(graph)

Arguments

graph

Input graph

Value

Named list of character vectors, the predecessors or the successors of each vertex.

Examples

G <- graph(list(A = c("B", "C"), B = "C", C = "A"))
predecessors(G)
successors(G)
