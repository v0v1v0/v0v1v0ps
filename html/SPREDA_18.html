kaplan.meier.location R Documentation

Kaplan-Meier Location

Description

Calculate the location of Kaplan-Meier estimator.

Usage

kaplan.meier.location(fit)

Arguments

fit

survfit object.

Value

xloc, yloc

x, y locations.

Author(s)

Yili Hong

Examples

require(survival)
fit <- survfit(Surv(time, status) ~ x, data = aml)
kaplan.meier.location(fit)
