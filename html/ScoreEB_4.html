multinormal R Documentation

Multivariate normal distribution

Description

Obtain P value with multivariate normal distribution.

Usage

multinormal(y,mean,sigma)

Arguments

y

column vector.
mean

arithmetic mean.
sigma

standard deviation.

Value

pdf_value

A vector of multivariate normal distribution density function.

Examples

data(pheno)
y <- pheno
mean <- 2.0
sigma <- 1.5
multinormal(y,mean,sigma)
