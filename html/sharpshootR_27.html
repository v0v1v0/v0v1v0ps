HHM R Documentation

Highland Meadows

Description

11 years of climate data from the Highland Meadows weather station, as maintained by CA DWR.

Usage

data("HHM")

Format

A data frame with 3469 observations on the following 12 variables.

station_id

a character vector

dur_code

a character vector

sensor_num

a numeric vector

sensor_type

a character vector

value

a numeric vector

flag

a character vector

units

a character vector

datetime

a POSIXct

year

a numeric vector

month

a factor with levels January February March April May June July August September October November December

water_year

a numeric vector

water_day

a numeric vector
