close_stream R Documentation

Close a Data Stream

Description

Close a data stream that needs closing (e.g., a file or a connection).

Usage

close_stream(dsd, ...)

Arguments

dsd

An object of class a subclass of DSD which implements a reset function.
...

further arguments.

Details

close_stream() is implemented for:

  • DSD

  • DSD_ReadCSV

  • DSD_ReadDB

  • DSD_ReadStream

  • DSF

  • DST_WriteStream
