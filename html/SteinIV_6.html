tr R Documentation

Trace of a matrix.

Description

Compute the trace of a square matrix.

Usage

  tr(X)

Arguments

X

Numeric: A square matrix.

Details

This computes the sum of the diagonal elements of a square matrix.

Value

numeric

A real number.
