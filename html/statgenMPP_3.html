barleyPheno R Documentation

Phenotypic data for awn length in barley

Description

Phenotypic data for awn length in barley

Usage

barleyPheno

Format

A data.frame with 916 rows and 3 columns.

genotype

Genotype

Awn_length

Awn_length in mm.

cross

The cross the genotype belongs to.

References
