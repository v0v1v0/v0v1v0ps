fa_filter R Documentation

fa_filter

Description

fasta filter by searching pattern

Usage

fa_filter(
  fasfile,
  pattern,
  by = "description",
  ignore.case = FALSE,
  outfile = NULL,
  type = "interleaved"
)

Arguments

fasfile

input fasta file
pattern

keyword for filter
by

one of 'description' and 'sequence'
ignore.case

logical
outfile

output file
type

one of 'interleaved' and 'sequential'

Value

BStringSet object
