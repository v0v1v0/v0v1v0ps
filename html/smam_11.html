estimate R Documentation

Estimate Result of smam Estimators

Description

'estimate' function returns the estimate result of 'smam::fitXXXX' from smam package.

Usage

estimate(x, ...)

## S3 method for class 'smam_mrh'
estimate(x, ...)

## S3 method for class 'smam_bmme'
estimate(x, ...)

Arguments

x
