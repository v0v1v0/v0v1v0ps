and R Documentation

Get Intersection Set for Sets

Description

Get intersection set for sets.

Usage

and(...)

a %and% b

Arguments

...

one or more objects
a

one object
b

the other object

Value

intersection elements

Examples

A <- c("a","b","c")
B <- c("a","b","c","d")
C <- c("a","e","h")
and(A, B)
and(A, B, C)
A <- c("a","b","c")
B <- c("a","b","c","d")
C <- c("a","e","h")
A %and% B
A %and% B %and% C
