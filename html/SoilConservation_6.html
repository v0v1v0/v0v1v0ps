KFactor R Documentation

Estimates soil erodibility.

Description

The function estimates the soil erodibility factor.

Usage

KFactor(df_kfactor)

Arguments

df_kfactor

Data to be analyzed.

Value

kfactor

Tabulated results of the k factor.
