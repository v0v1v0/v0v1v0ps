Description

Rename assays in a Seurat object

Usage

RenameAssays(
  object,
  assay.name = NULL,
  new.assay.name = NULL,
  verbose = TRUE,
  ...
)

Arguments

object

A Seurat object
assay.name

original name of assay
new.assay.name

new name of assay
verbose

Whether to print messages
...

Named arguments as old.assay = new.assay

Value

object with assays renamed

Examples

RenameAssays(object = pbmc_small, RNA = 'rna')
