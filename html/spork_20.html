as_plotmath.spork R Documentation

Convert Spork to Plotmath

Description

Converts spork to plotmath. See plotmath. Vectorized version of as_plotmath.spar.

Usage

## S3 method for class 'spork'
as_plotmath(x, ...)

Arguments

x

spork
...

passed to as_plotmath.spar

Value

plotmath
