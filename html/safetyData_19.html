sdtm_relrec R Documentation

sdtm_relrec

Description

Related Records data frame - One record per related record, group of records or datasets

Usage

sdtm_relrec

Format

a data frame with 234 rows and 7 columns. See https://github.com/SafetyGraphics/safetyData/blob/main/data-raw/sdtm/README.md for more details and full data specification. Columns:

STUDYID

text - Study Identifier

RDOMAIN

text - Related Domain Abbreviation

USUBJID

text - Unique Subject Identifier

IDVAR

text - Identifying Variable

IDVARVAL

text - Identifying Variable Value

RELTYPE

text - Relationship Type

RELID

text - Relationship Identifier
