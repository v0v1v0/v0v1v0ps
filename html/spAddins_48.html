insert_arrows R Documentation

Insert arrow (->, ->>, <-, <<-)

Description

RStudio add-in to insert either ->, ->>, <- or <<- at the cursor position.

Usage

rs_insert_arrow_rl()

rs_insert_arrow_rl2()

rs_insert_arrow_lr()

rs_insert_arrow_lr2()

Note

use hotkeys combination ALT + - to insert <- in RStudio.

See Also

Assignment operators assignOps.

Other 'Insert at cursor position' addins: insert_symbol_sequences

Examples

## Not run: 
\donttest{
library(spAddins)

rs_insert_arrow_rl()
## <-

rs_insert_arrow_rl2()
## <<-

rs_insert_arrow_lr()
## ->

rs_insert_arrow_lr2()
## ->>
}

## End(Not run)
