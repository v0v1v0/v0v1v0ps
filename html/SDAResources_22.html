classppsjp R Documentation

classppsjp data

Description

Joint inclusion probabilities for unequal probability sample without replacement from the population of classes in data classes.

Usage

data(classppsjp)

Format

This data frame contains the following columns:

class:

class ID number

class_size:

number of students in class

SelectionProb:

probability of being included in sample, \pi_i

SamplingWeight:

sampling weight w_i = 1/(\pi_i)

JtProb_1:

columns of joint inclusion probabilities, \pi_{1k}

JtProb_2:

columns of joint inclusion probabilities, \pi_{2k}

JtProb_3:

columns of joint inclusion probabilities, \pi_{3k}

JtProb_4:

columns of joint inclusion probabilities, \pi_{4k}

JtProb_5:

columns of joint inclusion probabilities, \pi_{5k}

References
