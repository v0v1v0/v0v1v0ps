stab_dist R Documentation

Stability Distance in AMMI

Description

Stability Distance of Genotypes in Additive ANOVA for Genotypes by Environment Interaction (GEI) model

Usage

stab_dist(.data, .y, .rep, .gen, .env, .m = 2)

## Default S3 method:
stab_dist(.data, .y, .rep, .gen, .env, .m = 2)

Arguments

.data

data.frame
.y

Response Variable
.rep

Replication Factor
.gen

Genotypes Factor
.env

Environment Factor
.m

No of PCs retained

Value

Stability Distance

data(ge_data)
YieldDist <-
     stab_dist(
            .data = ge_data

          , .gen  = Gen
