info_box R Documentation

Generates 'HTML' info box

Description

Generates 'HTML' info box

Usage

info_box(
  ...,
  icon = "envelope",
  class = "",
  class_icon = "bg-info",
  class_content = "",
  root_path = template_root()
)

Arguments

...

box content
icon

the box icon; default is "envelope", can be hidden by specifying NULL
class

class of the box container
class_icon

class of the icon
class_content

class of the box body
root_path

see template_root

Value

'HTML' tags

Examples


library(shiny)
library(shidashi)

info_box("Message", icon = "cogs")

info_box(
  icon = "thumbs-up",
  span(class = "info-box-text", "Likes"),
  span(class = "info-box-number", "12,320"),
  class_icon = "bg-red"
)

info_box("No icons", icon = NULL)
