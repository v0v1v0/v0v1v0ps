as.expression.plotmath R Documentation

Coerce Plotmath to Expression

Description

Coerces plotmath to expression by parsing as text.

Usage

## S3 method for class 'plotmath'
as.expression(x, ...)

Arguments

x

plotmath
...

ignored arguments

Value

expression
