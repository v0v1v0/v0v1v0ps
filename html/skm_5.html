col_rgn_val

Description

calculate colvec range = max - min value within limited range

Usage

col_rgn_val(u, wlmt)

Arguments

u

u: a numeric colvec
wlmt

wlmt: limit search on colvec on indices within wlmt

Value

vd max - min value in u within wlmt w.r.t to original index
