CL_FREQ R Documentation

Codelist Frequency

Description

Codelist Frequency, abbreviated as CL_FREQ. Version 2.1.

Usage

CL_FREQ

Format

A data frame with 34 rows and 5 variables:

id

A statistical code for describing data collection frequency.

name

The standard label of the code.

description

The description of the code.

name_locale

The language code of the name label.

description_locale

The language code of the description.

Details

The SDMX description: 'This code list provides a set of values indicating the frequency of the data (e.g. weekly, monthly, quarterly). The conceptfrequency may refer to various stages in the production process, e.g. data collection or data dissemination. For example, a time series could be disseminated at annual frequency but the underlying data are compiled monthly. The code list is applicable for all different uses offrequency.'
