plot_points R Documentation

Create map with points

Description

Create map for data.frame with points.

Usage

plot_points(
  df,
  value,
  lon = lon,
  lat = lat,
  palette = "viridis",
  legend_position = "bottomleft",
  crs = 4326,
  providers = c("CartoDB.Positron", "nlmaps.luchtfoto")
)

Arguments

df

data.framw with column for lon and lat
value

column in df
lon

column with lon
lat

column with lat
palette

color palette
legend_position

position for legend (default is "bottomleft")
crs

crs (default is 4326)
providers

providers to show. See 'leaflet::providers' for a list.

Value

leaflet map

Examples

## Not run: 
plot_points(Groningen, value = amount)

## End(Not run)
