new_sclr R Documentation

Create a new sclr object

Description

new_sclr creates the object sclr returns. is_sclr checks if the object is of class sclr.

Usage

new_sclr(fit, x, y, cl, mf, mt)

is_sclr(fit)

Arguments

fit

A list returned by sclr_fit.
x

Model matrix.
y

Model response.
cl

Call.
mf

Model frame.
mt

Model terms.

Value

sclr object
